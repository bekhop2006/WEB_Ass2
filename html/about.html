Frequently Asked Questions

What is this website about?

+

This is a news website created as part of our university assignment to showcase our web development skills. We built it using HTML5, CSS3, Bootstrap 5, and JavaScript to demonstrate modern web development practices.

Who created this website?

+

This website was created by Sariyev Maxot and Tanibergen Beknur from Astana IT University, Group SE-2406. We are passionate students learning web development and applying our knowledge in real projects.

What technologies were used?

+

We used HTML5 for structure, CSS3 for styling, Bootstrap 5 for responsive design, and JavaScript for interactive features including form validation, accordions, popups, and dynamic content.

How can I contact the team?

+

You can reach us through the Contact page on this website. We're always interested in feedback and collaboration opportunities. Our contact information is available in the Contact section.

Is this website responsive?

+

Yes! This website is fully responsive and works perfectly on desktop, tablet, and mobile devices. We used Bootstrap 5's grid system and responsive utilities to ensure optimal viewing experience across all screen sizes.
